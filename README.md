# SynFive Website

Sitio web oficial de SynFive.

## Stack

- PHP
- HTML
- TailwindCSS
- JavaScript

## Estructura

- `/includes` → lógica y helpers
- `/layout` → estructura visual
- `/components` → módulos reutilizables
- `/assets` → estilos, scripts, imágenes

## Features

- URLs limpias (.htaccess)
- SEO optimizado (meta, og, schema)
- Arquitectura modular escalable

## Deploy

Proyecto listo para producción en hosting con soporte PHP.
