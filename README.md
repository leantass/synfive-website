# SynFive Website

Sitio web oficial de SynFive.
Infraestructura digital para plataformas SaaS, sistemas distribuidos y backend cloud.

---

## Stack

- PHP
- HTML
- TailwindCSS
- JavaScript

---

## Arquitectura

- `/includes` → configuración, helpers y bootstrap
- `/layout` → head, header, navegación, footer
- `/components` → módulos reutilizables
- `/assets` → CSS, JS, imágenes

Arquitectura pensada para escalar y mantener consistencia en todo el sitio.

---

## Features

- URLs limpias (.htaccess)
- SEO completo (meta, canonical, OG, schema)
- Estructura modular reutilizable
- Helpers para rutas y assets
- Sistema base preparado para crecimiento
